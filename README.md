### 1. Markdown
>간단하게 문서를 작성할 수 있다. 확장명은 .md
>
- 사용하는 문법 정리
```
# : 제목, #수에 따라 다른 제목 양식 제공
1,2,3... : 목록(순서)
- : 목록(순서X)
```~~``` : Code block
`~` : Inline block
[] : Link
![] : Image
추가 문법은 다음 사이트 참고
[https://www.markdownguide.org/basic-syntax/#blockquotes-1]
```
